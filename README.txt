Instructions to run the scrabble_sentence_generator program:

1. Ensure that the word list file is in the same folder as the scrabble_sentence_generator.exe file and that it is in .txt format.
2. The word list file must be named "words_alpha.txt".
   2.1. If the word list file is not in the correct folder or if the format is incorrect, a prompt will appear where you can enter the file path of the word list file you wish 	to use.
   2.2. Example file paths:
       - Windows: "H:\Documents\Word files\words_alpha.txt"
       - Mac/Linux: "/Users/User/Documents/Word files/words_alpha.txt"
3. Double click on scrabble_sentence_generator.exe to run the program.
